		    3) UI Contents --> Checkers Board, 3 Buttons and 1 Label.

		    4) Button Hint --> Click on the checkers piece first, then click on Hint Button. It highlights the safest move in 'Magneta' color when needed to decide between left or right. But when you have a jump move or a move where only 1 direction movement is possible, it is highlighted in Green.

		    5) Button Resign  --> Ends the current game. This button is available when the game is launched.

		    6) Button New Game --> Allows user to play a game from start. This button is not available when the game launches because there is already a new game available at that instance. Once you click any of the other two buttons, this button will become available.

		    7) Message --> This label shows the current status of board with respect to player and move.

		    8) The available moves at any instance of the board is being printed out on the console for the user to verify the correctness of the program.
